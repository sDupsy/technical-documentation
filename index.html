HTML Basics

HTML (HyperText Markup Language) is the foundation of any webpage. It provides the structure and content.

Each webpage begins with a basic HTML skeleton that includes elements like head, title, and body.

<!DOCTYPE html> <html lang="en"> <head></head>
  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a>
CSS Styling

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It defines colors, spacing, fonts, and more.

You can apply CSS inline, internally with a style tag, or externally via a .css file.

h1 { color: blue; } p { font-size: 16px; }
  • Selectors: element, class, ID
  • Properties: color, margin, padding
  • Units: px, em, %, rem
JavaScript Functionality

JavaScript adds interactivity and behavior to web pages. It's the programming language of the web.

Common uses include reacting to user actions, validating input, or dynamically updating content.

console.log("Hello, world!"); document.getElementById("btn").onclick = () => alert("Clicked!");
  • Variables: let, const, var
  • Functions: function sayHi() {}
  • Events: click, hover, input
DOM Manipulation

The DOM (Document Object Model) is a tree-like structure representing the HTML of a page.

JavaScript can use the DOM to read, change, add, or remove elements from the page dynamically.

document.querySelector("h1").innerText = "Updated Title"; document.body.style.backgroundColor = "lightgray";
  • querySelector()
  • getElementById()
  • innerText / innerHTML
Putting It All Together

HTML, CSS, and JavaScript work together to create complete, functional web experiences.

HTML provides structure, CSS adds style, and JavaScript enables interaction.

<button id="myBtn">Click Me</button>
<script>
document.getElementById("myBtn").addEventListener("click", function() {
alert("Hello!");
});
</script>
  • Use semantic HTML
  • Separate style into CSS
  • Write clean, readable JS
